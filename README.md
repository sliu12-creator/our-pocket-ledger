# 🌊 我们的小账本 (Our Pocket Ledger)

More than a ledger.

A small project built step by step,
to record not only money,
but also everyday life.

---

## ✨ Current Features

- ✅ Local storage (no server required)
- ✅ Income & Expense management
- ✅ Two-level category system
- ✅ Monthly summary
- ✅ Recent records (Home)
- ✅ Full ledger view
- ✅ Record editing
- ✅ Mobile-first layout

---

## 🎯 Design Principles

- Mobile-first experience
- Clean and readable interface
- Keep the workflow simple
- Preserve data compatibility
- Separate business logic from UI rendering
- Prefer reusable components over duplicated code

---

## 🛠 Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- LocalStorage
